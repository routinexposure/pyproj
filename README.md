# Анализ частоты букв в тексте — Dash + Plotly

Этот проект представляет собой интерактивное веб-приложение на Dash, которое анализирует частоту появления букв в русском тексте на примере книги «Война и мир» Л. Н. Толстого.  
Приложение позволяет визуализировать распределение букв в нескольких форматах (столбцы, линия, площадь).

## Функциональность

Загрузка текста «Война и мир» и подсчёт частоты каждой буквы русского алфавита  
Фильтрация текста, учёт только букв русского алфавита  
Визуализация в виде:
столбчатой диаграммы  
линейного графика  
заполненной диаграммы  
Переключение темы оформления:
светлая (`plotly_white`)
тёмная (`plotly_dark`)
Полностью интерактивный график: подсказки, анимация, переходы


## Используемые технологии

1. Python 3
2. Dash — веб-фреймворк для аналитических приложений  
3. Plotly Graph Objects — построение графиков  
4. Counter (collections) — подсчёт частот букв  


## Структура проекта

project/

## Как запустить проект

1. Установка библиотек:

pip install dash plotly

Убеждаемся, что файл war_and_peace.txt находится в той же директории, что и script.py.
Запуск приложения:
python script.py

Перейдите по http://127.0.0.1:8050/
